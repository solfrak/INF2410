Pour le patron commande, les fichiers sont:
    button.h
    Command.h
    CopyCommand.h
    CutCommand.h
    DeleteCommand.h
    PasteCommand.h

Pour le patron mediator, les fichiers sont:

tests catch.h à l'aide d'un petit menu interactif.
